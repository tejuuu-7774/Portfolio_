# Portfolio & Project Showcase Platform
A full-stack portfolio platform built to showcase my projects, skills, and work in a clean and structured interface.

## Live Demo
https://tejaswinis-portfolio.vercel.app/

## Tech Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Serverless API Routes
- Nodemailer

## Features
- Dynamic project showcase with reusable components
- Clean and responsive UI with modern design system
- Contact form with email integration using Nodemailer
- Serverless backend for handling form submissions
- Fully deployed on Vercel

## Project Structure
- `app/` → Pages and API routes
- `components/` → Reusable UI components
- `lib/` → Data and utility functions
- `public/` → Static assets (images, resume)
